# Smooth Perspective

Client-side Fabric mod for smooth perspective switching.

## Features

- Smooth first/third-person transitions
- Configurable animation duration
- Configurable third-person distance with 0.1 precision
- Optional camera clip mode
- Cloth Config screen with Mod Menu integration

## Requirements

- Fabric Loader
- Fabric API
- Cloth Config
- Mod Menu is optional, but recommended for opening the config screen in-game
